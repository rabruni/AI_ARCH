# /prompts

LLM prompts, conversation templates, and AI interaction patterns.

## What Belongs Here

- System prompts for AI agents
- Conversation templates
- Few-shot examples
- Prompt engineering experiments
- Versioned prompt files for reproducibility

## What Does NOT Belong Here

- Source code (belongs in `/src`)
- Documentation about how prompts work (belongs in `/docs`)
- Runtime prompt generation code (belongs in `/src`)
- Actual conversation logs or outputs (should be gitignored)
